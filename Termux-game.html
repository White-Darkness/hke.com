Termux game is a Script where you can find all The Games that Are available for Termux as well as All the games that are available in Termux Repository. Developer have also included some games that are on GitHub or written in other languages other than bash like Hangman and 2048. This Script is not complete yet, so they still working on it so he will add more games in the Future.



REQUIREMENTS :
  • Internet
  • Storage permission

    • FEATURES :
  • [+] It is a collection of some amazing games for terminal
  • [+] It works both on Linux and Termux

    • INSTALLATION [Termux] :
    • apt update
    • apt upgrade
    • pkg install git
    • git clone https://github.com/khansaad1275/Termux-Games
    • cd Termux-Games
    • chmod +x *
    • ./install.sh
    • ./games.sh

      • Now enjoy from playing games...
